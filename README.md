# ✊ Rock Paper Scissors

A simple interactive Rock-Paper-Scissors game built with **HTML, CSS, and JavaScript**. Play against the computer with instant results and visual feedback.

---

## 🚀 Features

* Play against a random computer choice
* Instant result display (Win / Lose / Tie)
* Dynamic UI updates
* Visual feedback with color indicators

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

---

## ⚙️ How It Works

* Click on **Rock, Paper, or Scissors**
* Computer generates a random choice
* Result is displayed instantly
* UI updates with color feedback

---
